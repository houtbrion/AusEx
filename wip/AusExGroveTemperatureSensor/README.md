# AusEx library template for simple analog sensor
このライブラリは，
seeed studioのGroveシステムのいろいろなセンサ類のライブラリを
Adafruitのセンサ統合ライブラリと同じAPIに修正してみるプロジェクトの一部で
[Grove Temperature Sensor][ProductPage]用のドライバです．

[商品の技術情報][SeeedWiki]を見るとわかるように，商品の
リビジョンで特性が異なっており，本ドライバはv1.1とv1.2用です．


## ライセンスについて

このドライバは，[seeed studioのwiki][seeedwiki]のプログラムに
基づいていますが，ライセンス記載がないため，保留にします．
確認する必要がある人はseeed studioに聞いてください．
