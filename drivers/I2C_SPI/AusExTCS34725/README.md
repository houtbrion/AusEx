# TCS34725
このライブラリは，いろいろなセンサ類のライブラリを
Adafruitのセンサ統合ライブラリと同じAPIに修正してみるプロジェクトの一部で，
TCS34725-色センサ用ドライバです．

## ライセンスについて
ベースの[プログラム][github]はBSDライセンスとなっているので，
それを踏襲します．
